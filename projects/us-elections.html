US Presidential Elections Analysis

County-level analysis of 94,000+ voting records across 7 elections, exploring state loyalty, swing state dynamics, and the geographic realignment of American politics.

Tool
Tableau
Dataset
94,409 records
Period
2000–2024
Type
Exploratory Analysis

Key Findings

📊 2024 Broke the Popular Vote Pattern

Republicans won both the popular vote (91.4M) and Electoral College — the first time since 2004. Democrat vote share dropped from 52.3% to 48.4%.

🗺 The Blue Wall Crumbled

Pennsylvania, Michigan, and Wisconsin all flipped Republican in 2024 after going Democratic in 2020 — these three states were the decisive battlegrounds.

📈 Turnout Nearly Doubled in 24 Years

Total votes grew from 105M in 2000 to 199M in 2024, an 89% increase driven by both parties — Republicans alone gained 41M votes over the period.

📍 102 Counties Shifted Right in 2024

Between 2020 and 2024, 102 counties flipped from Democrat to Republican while only 15 went the other direction — a 7-to-1 ratio signalling broad geographic realignment.

Analysis Approach

1
Data Sourcing
Downloaded MIT Election Data + Science Lab's county-level presidential returns (2000–2024), containing candidate-party-county vote counts and FIPS codes.
2
Calculated Fields
Built Tableau calculated fields for Democrat Vote %, Republican Vote %, Winning Party per state/year, and Top N ranking filters.
3
Geographic Mapping
Created county and state choropleth maps showing winning party, with parameters controlling the election year displayed.
4
Loyalty Analysis
Scored states by how consistently they voted for the same party across all 7 elections — identifying 36 loyal states and 15 swing states.
5
Trend Analysis
Tracked vote share margins over time for 9 key battleground states, revealing long-term shifts like Florida's move from pure toss-up to solidly Republican.
6
Dashboard Assembly
Combined 14 worksheets into 2 interactive dashboards and 1 data story, with parameter actions for dynamic filtering.

Tableau Calculated Fields

Key calculated fields used to drive the analysis and interactivity:

// Winning Party per State per Year IF SUM([Democrat Votes]) > SUM([Republican Votes]) THEN "Democrat" ELSE "Republican" END // Democrat Vote % SUM([Democrat Votes]) / SUM([Total Votes]) // Top N Rank (for dynamic filtering) RANK_UNIQUE(SUM([candidatevotes]), 'desc')

What I Learned

Tableau Parameters

Using parameters for dynamic Top N filtering and year selection gives users control over the analysis without cluttering the view.

Geographic Storytelling

County-level maps reveal patterns invisible at the state level — like suburban shifts that drive statewide flips.

Data Story Format

Tableau's Story feature creates a guided narrative, but the web dashboard format reaches a wider audience without requiring Tableau access.

Political Data Nuance

Popular vote totals don't determine the winner — framing analysis around both popular vote and geographic distribution gives a more complete picture.
